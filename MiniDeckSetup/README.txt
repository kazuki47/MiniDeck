# MiniDeck インストールガイド

## 必要条件

- Windows 10以降
- .NET Framework 4.7.2以降
- 最小画面解像度: 1024x768

## インストール手順

1. ダウンロードしたMiniDeckSetup.msiファイルをダブルクリックします。
2. 表示される指示に従ってインストールを進めます。
3. インストール場所を選択するか、デフォルトのまま「次へ」をクリックします。
4. 「インストール」をクリックしてインストールプロセスを開始します。
5. インストールが完了すると、デスクトップとスタートメニューにショートカットが作成されます。

## プログラムの起動方法

以下のいずれかの方法でMiniDeckを起動できます：

- デスクトップのMiniDeckアイコンをダブルクリック
- スタートメニューから「MiniDeck」を選択
- インストール先フォルダの「MiniDeck.exe」を実行

## アンインストール方法

1. コントロールパネルから「プログラムと機能」を開きます。
2. リストから「MiniDeck」を見つけて選択します。
3. 「アンインストール」をクリックします。
4. 表示される指示に従ってアンインストールを完了します。

## トラブルシューティング

**Q: インストール中にエラーが発生する場合は？**  
A: 管理者権限でインストーラーを実行してください。インストーラーを右クリックして「管理者として実行」を選択します。

**Q: MiniDeckが起動しない場合は？**  
A: .NET Frameworkが正しくインストールされているか確認してください。必要に応じてMicrosoftの公式サイトから.NET Framework 4.7.2をダウンロードしてインストールしてください。

**Q: 画面がうまく表示されない場合は？**  
A: ディスプレイ設定でスケーリングを100%に設定してみてください。

## サポート
